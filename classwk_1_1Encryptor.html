Detailed Description

A class responsible for encrypting and decrypting data using AES-256-CBC.

The Encryptor class uses the OpenSSL library to encrypt and decrypt data with the AES-256-CBC algorithm. It generates an encryption key and an initialization vector (IV) from the provided password using SHA-512. This class requires the OpenSSL library.
